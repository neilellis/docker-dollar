docker-dollar
=============

A dockerized version of the DollarScript runtime. This is provided to make playing around with DollarScript a bit easier initially. Once you've got the hang of Dollar you'll probably want to [ ![Download](https://api.bintray.com/packages/neilellis/dollar/dollarscript/images/download.svg) ](https://bintray.com/neilellis/dollar/dollarscript/_latestVersion) and install directly.
